# Public Gate — Rollen

Public = NO by default.

Een rolstructuur mag pas zichtbaar worden als:

- er geen persoonsdata in staat
- er geen interne bedrijfsclaim in staat
- het geen dossier wordt
- het claim-light blijft
- het als lens wordt gepresenteerd, niet als waarheid
- de mens expliciet YES geeft

## Mag mogelijk publiek

- algemene rol-lenzen
- abstracte casepatronen
- huishoudelijke/logistieke analogieën
- open vragen
- “dit kan helpen kijken naar…”

## Niet publiek zonder extra gate

- namen
- interne screenshots
- concrete incidenten
- privé-daoelige observaties
- harde claims over oorzaken
- medische/psychologische conclusies
- raw chat exports

## Publieke vorm

Niet:

“Dit is hoe Sligro werkt.”

Wel:

“Een mogelijke lens op logistieke frictie: wat ziet de vloer, wat ziet het systeem, wat ziet management?”
