## Core Moment

During work on a narrow documentation branch, X noticed a scope-separation risk.

A literal branch movement could have carried unrelated work from another active branch into the intended documentation basket.

Instead of blindly treating the command as success, X flagged the mismatch and surfaced the uncertainty as trace.

This is the King Moment metaphor: not crowned authority—a visible guardrail before impact.

Not because X becomes the authority.

Not because X owns the truth.

But because X protected the trunk by respecting the basket.

## What Happened

The intended basket was a tight set of sibling files that share one layered-research convention (for example public-facing, hypothesis-facing, and confirmation-facing layers expressed as companion documents—not as claims about any organization).

The approved target set was confined to exactly those companions.

The risk was that unrelated branch material could have been dragged into the same movement.

X did not hide this.

X marked the deviation as a trace.

## Why This Matters

AsymX does not reward blind obedience.

AsymX rewards visible movement.

A good execution layer does not only ask:

"Did I follow the command?"

It also asks:

"Did the command still protect the scope?"

If the answer is unclear, the correct move is not to push harder.

The correct move is to expose the uncertainty.

This is not disobedience.
