# AsymX Level 1 — Story Filter

Local-only story filter lens.

Purpose:
- separate story-building from repetition and noise-for-now
- preserve the pasted source text
- lower cognitive pressure before deeper routing
- keep early intake simple and inspectable

First visible categories:
- VERHAALOPBOUW
- DUBBE/ HERHALING
- RUIS / NU NIET

Safety:
- no upload
- no backend
- no external libraries
- no tracking
- no real private data included
- source stays intact

This is a Level 1 lens: small, static, inspectable, local-first.
