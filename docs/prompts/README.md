# Prompts

Status: PUBLIC STRUCTURE

In AsymX, a prompt is an opening signal, not only an instruction.

A prompt may carry raw human movement, urgency, uncertainty, direction, emotion, or system intent. If a prompt is answered only on its surface wording, the response may become speculative or misaligned.

Important prompts should therefore preserve multiple layers:

1. Raw wording  
2. Loek translation  
3. Source translation  
4. Non-speculation boundary  
5. Desired response direction  
6. Routing  

## Core Rule

Do not answer only the surface words.

First translate the opening.

Then answer the mechanism.

## Routing

Public-safe prompt mechanisms may live here.

Private, sensitive, identifiable, or raw personal prompt material belongs in the local vault, not in public documentation.
