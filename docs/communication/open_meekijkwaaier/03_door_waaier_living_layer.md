# Door, Waaier, Living Layer

## Purpose

This document describes the three-layer structure of the Open Meekijkwaaier.

## Door

The door is a short human invitation.

It opens the meekijkruimte.

It does not carry the full value.

The door says:

- here is a visible field
- here is the context
- here is the next safe question
- correction is welcome

## Waaier

The waaier contains open examples, observations, questions and role views.

It spreads the field without forcing one conclusion.

The waaier can include:

- observations
- examples
- hypotheses
- questions
- patterns
- routes
- claim boundaries

## Living Layer

The living layer begins when people look, correct, add, park or route.

This is where value appears.

The living layer can contain:

- correction
- feedback
- updated view
- better labels
- parked uncertainty
- next safe movement

## Core Rule

The door is not the value.

The living meekijkruimte is the value.

The door only makes entry possible.
