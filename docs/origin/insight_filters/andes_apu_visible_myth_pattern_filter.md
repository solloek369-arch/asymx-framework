# ANDES / APU / VISIBLE MYTH PATTERN FILTER

Status: PUBLIC ORIGIN NOTE  
Claim level: INTERPRETIVE LENS / CREATIVE METAPHOR — ANDES-LAAG  
Public safety: Geen bewijsclaim, geen spirituele autoriteit, geen folklore-als-feit.

## CORE CORRECTION

Quechua belongs first to the mountain-field of the Andes: Cusco, Sacred Valley, Machu Picchu, peaks, terraces, stone, mist, sun, condor and Apu.

Amazonian Kichwa / Quechua-related traditions may touch the forest-river field, but they should not be collapsed into the same layer.

The Andes are not jungle.
The Amazon is not Machu Picchu.
Both may share patterns, but the source-fields stay separate.

## FUNCTION

This filter reads visible myth-patterns without turning them into proof.

It asks where similar forms appear across cultures:

- mountain guardians
- water beings
- small hidden beings
- threshold spirits
- warning animals
- saed stones
- mist gates
- voice-over-water stories
- disappearance / return patterns

## APU / MOUNTAIN GUARDIAN

Function: protection, height, boundary, elder presence, weather-body, community orientation.  
Ask: Is this mountain acting as orientation, or are we turning height into authority?  
Risk: Treating mountain language as proof.  
AsymX mapping: Apu = high constraint / guardian boundary / vertical field.  
KLEPPER: The mountain does not speak louder; it makes the field listen upward.
