# Root And Lens Map

Status: PUBLIC ORIGIN MAP  
Claim level: INTERPRETIVE / OPEN / NOT FINAL  
Public safety: No private source context included  

## Core Idea

AsymX mechanisms can be mapped through two different layers:

1. Roots  
2. Lenses / branches

Roots are deep orientation layers.

Lenses are functional filters that make mechanisms easier to see.

This is not a fixed doctrine.

This is not a spiritual ownership claim.

This is not a claim that any named person, tradition or culture endorses AsymX.
