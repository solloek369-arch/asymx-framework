# Retention Engine Foundation

## Core correction

This is not a chat dump.

This is not manual archiving.

This is not expanding the system.

This is retention:
existing information is sorted, ordered, connected and made traceable.

## Existing rule

"Wat zichtbaar genoeg is, hoeft nu niet opnieuw gedragen te worden."

"We gaan nu alleen nog maar kijken naar wat er al is."

## Meaning

The human operator does not carry the same visible information again.

The system carries it by creating:

- source pointer
- capsule
- label
- pressure level
- status
- visibility gate
- linked files
- linked engines
- linked project branches
- next safe step

## Retention flow

RAW SOURCE
→ INTAKE
→ SPLIT
→ LABEL
→ CAPSULE
→ LINK
→ INDEX
→ ROUTE
→ TRACE
