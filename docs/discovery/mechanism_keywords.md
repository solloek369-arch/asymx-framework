Status: PUBLIC DISCOVERY LAYER

## Core Rule

The search trail may be public.

The source context remains protected.

This layer contains keywords, pattern names, search intents and mechanism labels only.

- personal names

- private conversations

Create a public mechanism label instead.

Example:

Private source moment  
→ protected in local/private layer

Reusable mechanism  
→ may become public keyword

Public explanation  
→ mechanism only, no identifying context
