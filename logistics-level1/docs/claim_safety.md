# Claim Safety

## Do Not Say

- Sligro has proven ghost-picks.
- System X causes loss.
- Exact savings are proven.
- Management already knows this.

## Safer Language

- possible repeated cluster
- to be validated with internal data
- candidate for shadow run
- possible friction zone
- next step: validate location flow

The demo groups repeated deviations. It does not prove company-specific impact.
