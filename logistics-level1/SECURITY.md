# Security

This repository contains a static, local-only demonstration.

It does not contain:

- executable installers
- macros
- backend services
- databases
- npm dependencies
- external JavaScript libraries
- API calls
- tracking
- cookies
- authentication
- real company data

All CSV parsing happens inside the browser.
No data is uploaded or transmitted.

For internal company review, inspect `index.html` manually and run it offline if needed.
