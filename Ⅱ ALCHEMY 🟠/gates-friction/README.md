# Gates & Friction

This folder holds gate rules and scan tools for visible public material.

It does not prove that flagged files are wrong.

It marks where trace, claim-safety, soft contact or human gate may be needed.

Visibility is not automatically a mistake.  
Visibility creates responsibility.

## Core gates

- `TRACE_GATE.md` — no trace, no public; compost is not deletion; GitHub is the visible remainder after gate.
- `SOFT_CONTACT_RULE.md` — how much pressure may a public touch carry.
- `open_mechanics_map.md` — AsymX is a mechanics map, not a belief system.
- `PUBLIC_FRICTION_BOUNDARY.md` — pointer to the canonical public friction boundary inside `TRACE_GATE.md`.
- `spore-engine/` — scan tools for visible spores, thorns, compost and leaves.
