# Feedback Prompt Template

## Purpose

This template invites correction without turning the material into a conclusion.

## Questions

- What is recognizable?
- What is incorrect?
- What is missing?
- Which example would make this clearer?
- What should be parked?
- What is safe enough to test next?

## Closing Line

Correction is part of the meekijkruimte.

The next safe movement can stay small.
