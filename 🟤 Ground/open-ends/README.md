# Open ends

Open ends are not tasks by default.

They are unresolved pressure points, parked questions or unfinished traces that should not force immediate action.

This room keeps them visible without pushing them into public movement.

Rules:

- no automatic action
- no public claim
- no proof language
- no urgency unless a human gate marks it
- source and context stay intact

An open end may later become:

- a public leaf
- a mechanic note
- a route
- compost
- nothing
