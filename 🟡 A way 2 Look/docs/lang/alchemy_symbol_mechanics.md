## 1. Source caution

Alchemy texts are treated as historical and symbolic pattern fields.

Important boundary:

- The **Emerald Tablet** is treated as part of a **medieval Hermetic/alchemical textual tradition**, not as proof or direct “pharaonic” authority. Research summarized by **Oxford Cabinet** places the **textual trail** with **7th–9th century Arabic treatises**—useful as historical pattern context, not as ancient Egyptian evidence.
- **Paracelsus’ Tria Prima** is treated as a useful **three-part transformation metaphor** (often discussed in terms of sulphur / mercury / salt as volatile, fluid/changeable, and fixed/residue-like pattern poles), not as a physical or spiritual claim on the reader.
- **Modern Hermetic sources** such as **The Kybalion** (1908) are **not** treated as ancient texts or validators of AsymX.
- Any resemblance between alchemical language and AsymX structure is used **only** as routing metaphor.

Core rule:

```text
Resonance is not proof.
Metaphor is not authority.
Similarity is not ownership.
```

## 7. Symbol grammar

Symbols carry function.

| Symbol | Function |
| --- | --- |
| ⚫ | raw, source-dark, compost, not-yet-formed |
| ⚪ | web, visible field, public surface, reflected light |
| 🟤 | ground, soil, vault, archive, protected trace |
| 🟡 | signal, lens, looking, visible pattern |
| 🔵 | route, playfield, test, water-path |
| 🟢 | role, living choice, integration, safe handoff |
| 🔴 | fire, risk, release, pressure, stop/gate |
| 🟠 | flow, rhythm, timing, movement |
| 🌱 | mycelium link, relation without duplication |
| 📦 | park, later, held but not active |

Rule:

```text
One symbol should carry one primary function.
When a symbol carries too many functions, it stops routing and starts decorating.
```
