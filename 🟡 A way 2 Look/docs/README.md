# Docs — A way 2 Look

This docs room is intentionally small.

Only public-facing lenses and safety notes should remain here.  
Deep traces, rough chat md parked research belong in Ground.

## Public routes

| Route | Function |
|---|---|
| `discussions/` | public discussion starters and meekijkwaaier notes |
| `mechanics/` | looking mechanics and filter notes |
| `safety/` | claim-safety, boundary and public-gate notes |
| `lenses/` | lens material for reading signals |
| `lang/` | language and term notes |

## Not here anymore

The following types of material should not sit openly in this public docs room:

- raw AI-chat recovery traces
- unfinished case branches
- company-specific research material
- source dumps
- rough prompts
- origin material
- tentacle branches
- old experiments without a public summary

Those traces may remain preserved under Ground.

## Reading rule

Read public docs as lenses, not proof.

AI output is not source truth.  
Similarity is not ownership.  
Resonance is not evidence.  
A symbol is orientation, not authority.
