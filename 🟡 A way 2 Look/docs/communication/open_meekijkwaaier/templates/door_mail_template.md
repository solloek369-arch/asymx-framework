# Door Mail Template

## Purpose

This template opens a meekijkruimte.

It is generic.

It is not a sendable company-specific document.

## Template

Hi [PERSON/TEAM],

I am opening a small meekijkruimte around [FIELD/CONTEXT].

The goal is not to present a conclusion.

The goal is to make a few visible observations, examples and questions easy to review together.

You can find the meekijkruimte here:

[MEEKIJKRUIMTE LINK]

The next safe question is:

[NEXT SAFE QUESTION]

If something is incorrect, incomplete or unclear, correction is welcome.

Thanks.
