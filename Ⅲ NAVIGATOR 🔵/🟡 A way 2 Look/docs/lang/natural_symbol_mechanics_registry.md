# Natural Symbol Mechanics Registry

**Status:** INTERNAL / LANGUAGE ANCHOR  
**Public gate:** NOT PUBLIC BY DEFAULT

---

## Core rule

Natural symbols are used only when they carry a visible mechanic.  
They are routing markers, not decoration and not proof.

Natuurlijke symbolen worden alleen gebruikt wanneer ze een zichtbare mechaniek dragen.  
Ze zijn routingtekens, geen decoratie en geen bewijs.

---

## Why this layer exists

AsymX may use natural symbols only when the symbol compresses an already visible mechanic.

The symbol is not chosen because it looks beautiful.  
The symbol is not used as spiritual proof.  
The symbol does not validate the system.

The symbol helps route attention.

A natural symbol enters the public layer only when this chain is clear:

```text
natural mechanic → AsymX mechanic → symbol → route
```

For aligned routing shorthand and mechanic-linked rows, see also §**0.5** in [`symbol_grammar.md`](symbol_grammar.md).
