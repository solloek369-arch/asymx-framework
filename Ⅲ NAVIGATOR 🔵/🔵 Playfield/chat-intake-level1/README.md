# AsymX Level 1 — Chat Intake Vault

Local-only intake lens for AI chats.

Purpose:
- preserve source text intact
- treat analysis as metadata
- separate story-building, repetition, and noise-for-now
- support safe compression of chats into later capsules

Safety:
- no upload
- no backend
- no external libraries
- no tracking
- no real private data included
- source stays inta

This is a Level 1 lens: small, static, inspectable, local-first.
