# Afterwords

Status: PUBLIC STRUCTURE

In AsymX, an afterword is a landing trace.

It records what became visible after a prompt, movement, answer, action, correction, or system event.

An afterword is not only a summary. It can preserve:

- what opened
- what changed
- what became clearer
- what needs correction
- what can become public
- what must remain private
- what new wording, path, or rule appeared

## Core Rule

A prompt opens a field.

An afterword records what the field returned.

## Routing

Public-safe afterwords may live here.

Private, sensitive, identifiable, or raw personal reflections belong in the local vault, not in public documentation.
