# Language Access Layer

This folder contains access languages for AsymX readability.

Access languages are not source layers.

The Dual Source structure remains separate:

- Dutch / `nl`
- Quechua-Kichwa reflection / `qu`

Access languages help people read, verify, translate and respond across language fields.

## Language Access Correction — Three World Powers

This language layer now explicitly includes three major world-power language zones:

- America / English / `en`
- China / Chinese / `zh`
- Russia / Russian / `ru`

The order is alphabetical according to Dutch:

Amerika.  
China.  
Rusland.

This is not a political endorsement.  
This is not a hierarchy of human value.  
This is an access correction.

The absence of Russian (`ru`) was a visible gap in the language layer.

Language is not agreement.  
Language is access.
