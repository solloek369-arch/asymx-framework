# Shadow Run Book

Use this process for an internal, low-risk review.

1. Export manco and short-pick events for one week.
2. Remove personal data.
3. Use only location, time, event_type, item, and route fields.
4. Open `index.html` locally.
5. Load the CSV.
6. Review possible repeated deviation clusters.
7. Validate top locations on the floor.
8. Discuss a possible minimal pilot.

The shadow run does not require operational changes. It is a visibility step, not a conclusion.
