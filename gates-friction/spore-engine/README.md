# Spore Engine

**Status:** gate / trace engine  
**Function:** read what became visible and route it without panic.

The Spore Engine does not assume that visible material is wrong.

It asks:

- What became visible?
- Is it leaf, seed, spore, compost or thorn?
- Can it remain public?
- Does it need a label?
- Does it need a human gate?
- Should it return to vault/source?

## Natural statuses

| Status | Meaning |
|---|---|
| LEAF | safe visible public surface |
| SEED | small public beginning, not yet full route |
| SPORE | signal that spread or appeared; follow before acting |
| COMPOST | valuable material that should return to source/vault |
| THORN | risk-bearing visible material; gate before movement |

## Rule

> Visibility is not automatically a mistake.  
> Visibility creates responsibility.

## Core AsymX line

Make it visible without making it heavier.
