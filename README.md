# AsymX Framework 
Coding from the purest structural base currently reached (19-3-2026).

A simple, original framework for **stable rivalry** (+ vs −) and better thinking.

## 🧠 What is AsymX?
Created by Loek Verdonk.

Four easy building blocks:
- ⚫ field (completely empty and passive)
- **+** = appearance (new idea)
- **−** = disappearance (counter-argument)
- **X** = stable intersection (when + and − meet)

From this, paths grow and sometimes create rivalry that doesn't have to be solved immediately.

## ⚙️ What you find here
- A working simulation (you see live how + and − make X)
- Perfect for AI reasoning (AI can now hold two ideas at the same time without choosing)
